# Titanic-Dataset
# 🚢 Titanic Survival Prediction using Logistic Regression

## 📌 Overview

This project aims to predict passenger survival on the Titanic using **Machine Learning**, specifically the **Logistic Regression** algorithm.

The analysis focuses on identifying key factors that influenced survival, such as:

* Age
* Gender
* Passenger Class (Pclass)
* Number of siblings/spouses (SibSp)
* Number of parents/children (Parch)
* Fare

---

## 📊 Dataset

The project uses the well-known **Titanic Dataset**, which contains demographic and travel information about passengers.

**Source:**

* Kaggle Titanic Dataset

---

## 🔍 Workflow

### 1. Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Dropping irrelevant features

---

### 2. Exploratory Data Analysis (EDA)

* Data distribution analysis
* Identifying relationships between features and survival
* Visualization to uncover patterns

---

### 3. Feature Engineering

* Selecting and transforming important features
* Improving model performance

---

### 4. Model Building

* Applying **Logistic Regression**
* Splitting data into training and testing sets

---

### 5. Model Evaluation

The model was evaluated using:

* Accuracy
* Confusion Matrix
* Precision & Recall

---

## 📈 Results

The model achieved a reasonable accuracy in predicting survival. Key insights include:

* **Females** had a higher survival rate
* Passengers in **1st class** were more likely to survive
* Age had a noticeable impact on survival probability
